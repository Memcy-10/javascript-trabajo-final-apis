# Frankfurter App 💱

Aplicación web para consultar tasas de cambio usando la API pública de [Frankfurter](https://api.frankfurter.app).

## API utilizada

**Frankfurter** — `https://api.frankfurter.app`

API gratuita, sin registro y sin API key. Publica las tasas de cambio del Banco Central Europeo para 33 monedas principaless
